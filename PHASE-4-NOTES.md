## API Endpoints
- GET /api/attendance/today - Today's attendance
- GET /api/attendance/date/:date - By date
- GET /api/attendance/member/:memberId - History
- POST /api/attendance - Mark attendance
- GET /api/attendance/summary/:memberId - Monthly
- GET /api/leave/member/:memberId - Leaves
- POST /api/leave - Mark leave
- DELETE /api/leave/:leaveId - Cancel
